eNtweniChat - A Real-Time Chat App (enhanced)

eNtweniChat is a real-time chat application that allows users to connect, exchange messages, and navigate between different screens. It utilizes modern technologies to provide a robust and efficient communication experience.

Description

The project leverages core components like configuration management, navigation management, server communication, and message handling. The backend is built with FastAPI and websockets for real-time functionality, while a SQLite database manages user data and message history.

Key Features

Backend

  • **FastAPI Backend:** Provides a performant framework for websockets and user interactions.
  • **WebSocket Communication:** Enables real-time message exchange between users.
  • **User Registration:** Users can register their phone numbers to participate in chats.
  • **Message Sending & Receiving:** Users can send and receive messages from others.
  • **Message Persistence:** Messages are stored in a SQLite database for reliable retrieval.
  • **New Message Retrieval:** Efficiently fetch new messages you haven't received yet.
  • **Message Read Status:** Tracks whether a user has read a message.

Frontend

  • **Configuration Management:** Centralizes app settings for easy modification.
  • **Navigation Management:** Facilitates navigation between screens, including improved back navigation handling.
  • **Server Communication:** Establishes and manages a WebSocket connection for real-time communication.
  • **Message Handling:** Efficiently processes incoming messages to update the chat UI.
